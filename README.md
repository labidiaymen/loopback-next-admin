## How to run the project

This project is composed of two parts:
Client that hold the front-end logic Angular/React/Vue projects
Api that hold all backend logic

## Commands

To run the front
```
 $ cd client/
 $ npm i
 $ npm start
 ```

The backend :
````
cd api/
npm i
npm start
````
